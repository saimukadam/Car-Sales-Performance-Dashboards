# Car Sales Performance Dashboard

This project showcases a dynamic dashboard built using Looker Studio to analyze car sales data across brands, regions, fuel types, and transmission types.

## 📊 Metrics
- Revenue
- Profit
- Units Sold
- Discount Amount

## 📍 Filters
- Brand
- Region
- Fuel Type
- Transmission
- Date Range
